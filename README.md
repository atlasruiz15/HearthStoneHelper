HearthStoneHelper
=================

(Spanish / Español)

¿Qué es HearthStoneHelper?
Es una herramienta web para llevar el seguimiento de tu mazo mientras juegas al HearthStone.

¿Qué hace HearthStoneHelper?
Con HearthStoneHelper puedes importar mazos creados desde la página "HearthHead.com", bien desde el texto exportado en HTML o desde la url del "DeckBuilder".
Una vez importados, puedes guardarlos para su posterior uso, si así lo deseas.
HearthStoneHelper te mostrará tu mazo de cartas, junto con la cantidad de cada una de ellas. A medida que vayas descartando cartas en tu juego, puedes ir tachándolas en tu mazo, en la herramienta, y así llevar un seguimiento de las cartas jugadas.
Al terminar la partida, puedes reiniciar el mazo a su estado inicial.

¿Cómo se usa?
Es una página web, por lo que sólo tienes que abrirla en tu navegador y emplear los controles/botones que allí veas.
En principio es compatible con las últimas versiones de todos los navegadores, y puede ser empleado en cualquier plataforma, ya sea en computadores personales o tabletas (bajo Windows/MacOS/iOS).
En el caso de dispositivos táctiles, las cartas del mazo se activan/desactivan mediante deslizamientos del dedo de a izquierda o derecha; En el caso de computadores, pulsando con el botón izquierdo/derecho del ratón.

¿Qué más debo saber de HearthStoneHelper?
Pues poco más. Surgió como un proyecto personal, por lo que no esperes encontrar un programa perfecto tanto en diseño como en programación. Por ahora, cumple su cometido.
Al ser una página web, si juegas a pantalla completa, no podrás tenerla sobre el juego, por lo que quizás debas emplear otro dispositivo como apoyo para usarla.
La herramienta está pensada para jugar con mazos en español o inglés, por lo que es "bilingüe", alternando a tu gusto mediante botones en el interfaz.

¡Espero que la encuentres útil! ^^

PD: HearthStoneHelper usa parte de código fuente del Deck Builder de HearthHead, para importar mazos desde url. Todo el crédito por este código le corresponde a sus propietarios originales.

(English / Inglés)

What is HearthStoneHelper?
It is a tool to keep track of your played cards while playing HearthStone.

What does HearthStoneHelper?
You can import decks from "HearthHead.com", either from HTML export text or an url from its Deck Builder tool.
Once you import a deck, you can store it for a later use, if you want.
HearthStoneHelper will show you your deck, among with the count of each card. As you draw cards while playing, you can delete them in the tool, so you can track your played cards.
When you finish a game, you can reset the deck to the initial state and so on.

How is it used?
It is a web page, so you only need to open it on your browser and to use the buttons and controls there.
It should be support last versions of all browsers and it can be used under any platform and architecture, either personal computers or portable devices (under Windows, MacOS, Linux, iOS...)
For touch devices, the cards in the deck are activated / deactivated using left / right swipes with the finger; For computers, they are activated / deactivated with left / right mouse clicks.

Anything more to know about HearthStoneHelper?
Not much.

It was created as a personal project, so do not expect to find a perfect designed / programmed thing. It does the job for now. 
Being a web page, you can not see it when playing at full screen. So you may need to use it on other device as support. 
The tool is intended to be used when you play Spanish or English game clients, so it is "bilingual", and you can toggle languages as you like using the right buttons on the UI.

I hope you find it useful! ^^

PD: HearthStoneHelper uses part of HearthHead Deck Builder source to import decks from url. All credit for it goes to the creators of this source.
